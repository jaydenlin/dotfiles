# Dotfiles

## Installation

### dotfiles
```
git clone https://github.com/jaydenlin/dotfiles.git && cd dotfiles && source install.sh
```
### VIM
```
git clone git://github.com/gmarik/vundle.git ~/.vim/bundle/vundle
vim
:PluginInstall!
```
